Experiences

2012Engineer for alcotra project “myMed” in INRIA Sophia-Antipolis LogNet Team
A “social web apps” project involving University of Torino. The challenge was to cluster 40 computers across the two parties and several distant places in the department (St Martin de vésubie, Menton,...). Each node exposes a classic static server (web applications), and a backend server for accessing database (NoSQL Cassandra). I have worked on both parts, we made minimal query language that can be used on key-value stores. I also started a more interactive content-based pubsub over HTTP (websockets, or by default in-database notifications) to replace current mail alerts.
More investigations dealt with load balancing and optimization of requests (geographical or latencies-based)

2011Engineer for ADT project “Sensas” in INRIA Nancy Madynes Team
In the SensMgt group, we focused on sensor networks management using a widespread OS Contiki, and its network simulator Cooja. We worked on porting a snmp server implementation to SensLab sensors. I made a browser-based (Firefox) network manager for snmp:// uri scheme, similarly to Copper extension (coap is the equivalent of HTTP in sensor networks), and started to port them on Chrome (UDP handling in extensions is far easier).
Our next task was a smart piggybacking for propagating some pending management data inside ipv6 hop-by-hop headers (full report).

2010Engineer with Fireflies-rtls grant in INRIA Nancy Madynes Team
We implemented geopositioning and asset management applications based on TI SoC wireless sensors and their ZigBee stack. Other related works included a distance-to-anchors propagation algorithm (CFIP 2011 submission), a SNMP proxy (PySNMP extension) for managing and caching sensors data, a Python serial interactive shell for sending commands..

2009 (03-09)Graduate Internship in INRIA Nancy Madynes Team
Geolocation system in WiFi environment using PlaceLab with different strategies e.g. least squares multilateration, signal fingerprinting.. (presentation, state of the art).
