2012Engineer for Alcotra project “myMed” in INRIA Sophia-Antipolis LogNet Team

2011Engineer for ADT project “Sensas” in INRIA Nancy Madynes Team

I made a browser-based (Firefox) network manager for snmp:// uri scheme, similarly to Copper extension (coap is the equivalent of HTTP in sensor networks).
Our next task was a smart piggybacking for propagating some pending management data inside ipv6 hop-by-hop headers (full report).

Other related works included a distance-to-anchors propagation algorithm (CFIP 2011 submission), a SNMP proxy (PySNMP extension) for managing and caching sensors data, a Python serial interactive shell for sending commands..
